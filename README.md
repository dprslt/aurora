# Aurora

## Prérequis

### Packages python : 
```
python3-rpi.gpio
python3-numpy
```

### Librairies Neo pixels

https://github.com/jgarff/rpi_ws281x

Il faut installler les dépendances et cette librairie en utilisant python 3

## Utilisation

Instalation et lancement du daemon

```
./install.sh
```
